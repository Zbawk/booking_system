# Booking System
* run 'composer install'
* run 'npm install'
* run 'npm run dev'

## database
* run 'cp .env.example .env'
* edit the .env file to 'booking_system'
* create 'booking_system' as db in php my admin
* run 'php artisan migrate'

## for the app key
* run 'php artisan key:generate'
* run 'php artisan serve'
